# LeRobot AlohaMini 常用指令

## 一、校准指令

### Mac 端 - 校准主臂

```bash
# 校准左主臂
lerobot-calibrate \
  --teleop.type=so101_leader \
  --teleop.port=/dev/cu.usbmodem5AE60814681 \
  --teleop.arm_profile=so-arm-5dof \
  --teleop.id=left_leader

# 校准右主臂
lerobot-calibrate \
  --teleop.type=so101_leader \
  --teleop.port=/dev/cu.usbmodem5AE60528341 \
  --teleop.arm_profile=so-arm-5dof \
  --teleop.id=right_leader
```

### 树莓派端 - 校准从臂

```bash
# SSH 到树莓派
ssh ubuntu@192.168.3.33

# 激活环境并校准
conda activate lerobot_alohamini
cd ~/lerobot_alohamini
lerobot-calibrate --robot.type=lekiwi
```

---

## 二、运行指令

### Mac 端 - 启动遥操作（一键脚本）

```bash
# 方法 1：使用一键启动脚本（推荐）
cd ~/Repository/lerobot_alohamini
./start_teleoperate.sh

# 方法 2：手动启动
conda activate lerobot_alohamini
python examples/alohamini/teleoperate_bi.py \
  --remote_ip 192.168.3.33 \
  --leader_id so101_leader_bi \
  --arm_profile so-arm-6dof
```

### 树莓派端 - 启动从臂服务

```bash
# SSH 到树莓派
ssh ubuntu@192.168.3.33

# 激活环境并启动服务
conda activate lerobot_alohamini
cd ~/lerobot_alohamini
python -m lerobot.robots.alohamini.lekiwi_host \
  --arm_profile so-arm-5dof
```

## 三、键盘控制说明

- **W/S**: 前进/后退
- **A/D**: 左转/右转
- **Z/X**: 左移/右移
- **U/J**: 升降轴上升/下降
- **R/F**: 加速/减速
- **Q**: 退出程序

## 四、常用调试指令

### 查看串口设备

```bash
# Mac
ls /dev/cu.usbmodem*

# 树莓派
ls /dev/ttyACM*
```

### 测试网络连接

```bash
# Mac 端测试树莓派连接
ping 192.168.3.33

# 测试 ZMQ 端口
nc -zv 192.168.3.33 5555
nc -zv 192.168.3.33 5556
```

### 查看校准文件

```bash
# Mac 端
ls ~/.cache/huggingface/lerobot/calibration/teleoperators/so_leader/

# 树莓派端
ls ~/.cache/huggingface/lerobot/calibration/robots/lekiwi/
```

## 五、完整启动流程

1. **树莓派端**（先启动）
   ```bash
   ssh ubuntu@192.168.3.33
   conda activate lerobot_alohamini
   cd ~/lerobot_alohamini
   python -m lerobot.robots.alohamini.lekiwi_host --arm_profile so-arm-5dof
   ```

2. **Mac 端**（后启动）
   ```bash
   cd ~/Repository/lerobot_alohamini
   ./start_teleoperate.sh
   ```

3. **开始遥操作**
   - 移动主臂，从臂会跟随
   - 使用键盘控制底盘和升降轴
   - 按 Q 或 Ctrl+C 退出
